SVM is a famous supervised machine learning algorithm used for classification as well as regression algorithms. However,  mostly it is preferred for classification algorithms. It basically separates different target classes in a hyperplane in n-dimensional or multidimensional space. The main motive of the SVM is to create the best decision boundary that can separate two or more classes(with maximum margin) so that we can correctly put new data points in the correct class.  Wait! Why is it known as SVM? Because It chooses extreme vectors or support vectors to create the hyperplane, that’s why it is named so. In the below sections let's understand in more detail.
